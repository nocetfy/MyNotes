                          # 单调栈

                          ​ 如果问题要求上一个最大(小)的元素，下一个最大(小)的元素，就可以使用单调栈解决。

                          ​ 核心原则：即使去掉无用元素，保证栈中元素有序。

                          • 单调递增栈，利用波谷剔除栈中的波峰，留下波谷；
                          • 单调递减栈，利用波峰剔除栈中的波谷，留下波峰。

                          # 经典问题

                          # 1. 模板题

                          示例 1:

                          输入： 5
1 4 2 3 5
输出： 2 5 4 5 0

                          思路：

                          ​ 使用单调递减栈保存下标，如果遇到比它大的数，则出栈，记录此下标遇到的下一个数的下标。

                          class Solution {
    int[] next(int[] array) {
        int[] res = new int[array.length];
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < array.length; i++) {
            while (!stack.isEmpty() && array[i] > array[stack.peek()]) {
                res[stack.pop()] = i + 1;
            }
            stack.push(i);
        }
        return res;
    }
}

                          # 155. 最小栈open in new window

                          设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

                          实现 MinStack 类:

                          • MinStack() 初始化堆栈对象。
                          • void push(int val) 将元素 val 推入堆栈。
                          • void pop() 删除堆栈顶部的元素。
                          • int top() 获取堆栈顶部的元素。
                          • int getMin() 获取堆栈中的最小元素。

                          示例 1:

                          输入：
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[-2],[0],[-3],[],[],[],[]]

输出：
[null,null,null,null,-3,null,0,-2]

解释：
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin();   --> 返回 -3.
minStack.pop();
minStack.top();      --> 返回 0.
minStack.getMin();   --> 返回 -2.

                          思路：

                          ​ 使用一个辅助栈保存每次push操作时的最小值，当出栈时，辅助栈也跟着出栈。

                          class MinStack {
    Deque<Integer> xStack = new ArrayDeque<>();
    Deque<Integer> minStack = new ArrayDeque<>(List.of(Integer.MAX_VALUE));

    public void push(int x) {
        xStack.push(x);
        minStack.push(Math.min(minStack.peek(), x));
    }

    public void pop() {
        xStack.pop();
        minStack.pop();
    }

    public int top() {
        return xStack.peek();
    }

    public int getMin() {
        return minStack.peek();
    }
}

                          # 739. 每日温度open in new window

                          给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指在第 i天之后，才会有更高的温度。如果气温在这之后都不会升高，请在该位置用 0 来代替。

                          示例 1:

                          输入: temperatures = [73,74,75,71,69,72,76,73]
输出: [1,1,4,2,1,1,0,0]

                          示例 2:

                          输入: temperatures = [30,40,50,60]
输出: [1,1,1,0]

                          示例 3:

                          输入: temperatures = [30,60,90]
输出: [1,1,0]

                          思路：使用单调递减栈，存储下标，注意遍历比较时需要使用while然后根据下标计算距离。

                          class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        int length = temperatures.length;
        int[] res = new int[length];
        // 单调递减栈，存下标
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < length; i++) {
            // 每个栈顶的值都比较是否找到了更大的值，如果是，将取出的下标进行赋值
            while (!stack.isEmpty() && temperatures[i] > temperatures[stack.peek()]) {
                int index = stack.pop();
                res[index] = i - index;
            }
            stack.push(i);
        }
        return res;
    }
}

                          # 496. 下一个更大元素 Iopen in new window

                          nums1 中数字 x 的 下一个更大元素 是指 x 在 nums2 中对应位置 右侧 的 第一个 比 x 大的元素。

                          给你两个 没有重复元素 的数组 nums1 和 nums2 ，下标从 0 开始计数，其中 nums1 是 nums2 的子集。

                          对于每个 0 <= i < nums1.length ，找出满足 nums1[i] == nums2[j] 的下标 j ，并且在 nums2 确定nums2[j] 的 下一个更大元素 。如果不存在下一个更大元素，那么本次查询的答案是 -1 。返回一个长度为 nums1.length 的数组 ans 作为答案，满足 ans[i] 是如上所述的 下一个更大元素 。

                          示例 1：

                          输入：nums1 = [4,1,2], nums2 = [1,3,4,2].
输出：[-1,3,-1]
解释：nums1 中每个值的下一个更大元素如下所述：
- 4 ，用加粗斜体标识，nums2 = [1,3,4,2]。不存在下一个更大元素，所以答案是 -1 。
- 1 ，用加粗斜体标识，nums2 = [1,3,4,2]。下一个更大元素是 3 。
- 2 ，用加粗斜体标识，nums2 = [1,3,4,2]。不存在下一个更大元素，所以答案是 -1 。

                          示例 2：

                          输入：nums1 = [2,4], nums2 = [1,2,3,4].
输出：[3,-1]
解释：nums1 中每个值的下一个更大元素如下所述：
- 2 ，用加粗斜体标识，nums2 = [1,2,3,4]。下一个更大元素是 3 。
- 4 ，用加粗斜体标识，nums2 = [1,2,3,4]。不存在下一个更大元素，所以答案是 -1 。

                          思路：

                          ​ 单调栈，将nums2中所有元素都保存下一个最大元素，然后从哈希表取即可。

                          class Solution {
    public int[] nextGreaterElement(int[] nums1, int[] nums2) {
        int[] res = new int[nums1.length];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new ArrayDeque<>();
        Map<Integer, Integer> map = IntStream.range(0, nums1.length)
                .boxed()
                .collect(Collectors.toMap(i -> nums1[i], Function.identity()));
        for (int x : nums2) {
            while (!stack.isEmpty() && stack.peek() < x) {
                int index = stack.pop();
                if (!map.containsKey(index)) {
                    continue;
                }
                res[map.get(index)] = x;
            }
            stack.push(x);
        }
        return res;
    }
}

                          # 503. 下一个更大元素 IIopen in new window

                          给定一个循环数组 nums （ nums[nums.length - 1] 的下一个元素是 nums[0] ），返回 nums 中每个元素的 下一个更大元素 。

                          数字 x 的 下一个更大的元素 是按数组遍历顺序，这个数字之后的第一个比它更大的数，这意味着你应该循环地搜索它的下一个更大的数。如果不存在，则输出 -1 。

                          示例 1:

                          输入: nums = [1,2,1]
输出: [2,-1,2]
解释: 第一个 1 的下一个更大的数是 2；
数字 2 找不到下一个更大的数； 
第二个 1 的下一个最大的数需要循环搜索，结果也是 2。

                          示例 2:

                          输入: nums = [1,2,3,4,3]
输出: [2,3,4,-1,4]

                          思路：

                          ​ 遍历数组两次，即可得到循环数组中所有元素的最大值，所以需要把下标取模特殊进行处理即可。使用单调递减栈保存数组的下标。

                          class Solution {
    public int[] nextGreaterElements(int[] nums) {
        int[] res = new int[nums.length];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < nums.length * 2; i++) {
            while (!stack.isEmpty() && nums[stack.peek() % nums.length] < nums[i % nums.length]) {
                res[stack.pop() % nums.length] = nums[i % nums.length];
            }
            stack.push(i);
        }
        return res;
    }
}

                          # 2454. 下一个更大元素 IVopen in new window

                          给你一个下标从 0 开始的非负整数数组 nums 。对于 nums 中每一个整数，你必须找到对应元素的 第二大 整数。

                          如果 nums[j] 满足以下条件，那么我们称它为 nums[i] 的 第二大 整数：

                          • j > i
                          • nums[j] > nums[i]
                          • 恰好存在 一个 k 满足 i < k < j 且 nums[k] > nums[i] 。

                          如果不存在 nums[j] ，那么第二大整数为 -1 。

                          • 比方说，数组 [1, 2, 4, 3] 中，1 的第二大整数是 4 ，2 的第二大整数是 3 ，3 和 4 的第二大整数是 -1 。

                          请你返回一个整数数组 answer ，其中 answer[i]是 nums[i] 的第二大整数。

                          示例 1：

                          输入：nums = [2,4,0,9,6]
输出：[9,6,6,-1,-1]
解释：
下标为 0 处：2 的右边，4 是大于 2 的第一个整数，9 是第二个大于 2 的整数。
下标为 1 处：4 的右边，9 是大于 4 的第一个整数，6 是第二个大于 4 的整数。
下标为 2 处：0 的右边，9 是大于 0 的第一个整数，6 是第二个大于 0 的整数。
下标为 3 处：右边不存在大于 9 的整数，所以第二大整数为 -1 。
下标为 4 处：右边不存在大于 6 的整数，所以第二大整数为 -1 。
所以我们返回 [9,6,6,-1,-1] 。

                          示例 2：

                          输入：nums = [3,3]
输出：[-1,-1]
解释：
由于每个数右边都没有更大的数，所以我们返回 [-1,-1] 。

                          提示：

                          • 1 <= nums.length <= 10^5
                          • 0 <= nums[i] <= 10^9

                          思路：

                          ​ 将首次遇到比它大的数先出栈，然后保存到优先级队列中去，队列中最小的元素优先进行比较是否再次遇到了比它大的元素，注意队列保存的是元素的下标，所以需要使用逆序。

                          class Solution {
    public int[] secondGreaterElement(int[] nums) {
        int length = nums.length;
        int[] res = new int[length];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new ArrayDeque<>();
        PriorityQueue<Integer> second = new PriorityQueue<>(Comparator.reverseOrder());
        for (int i = 0; i < length; i++) {
            while (!second.isEmpty() && nums[second.peek()] < nums[i]) {
                res[second.poll()] = nums[i];
            }
            while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
                second.offer(stack.pop());
            }
            stack.push(i);
        }
        return res;
    }
}

                          思路：

                          ​ 使用一个双向队列来进行维护，将第一次遇到比它更大的数弹出栈，按出栈的反顺序(即最开始入栈的顺序)加入双向队列，然后在队列中判断是否又遇到了更大的数。

                          class Solution {
    public int[] secondGreaterElement(int[] nums) {
        int length = nums.length;
        int[] res = new int[length];
        Arrays.fill(res, -1);
        Deque<Integer> stack = new ArrayDeque<>(), second = new ArrayDeque<>(), temp = new ArrayDeque<>();
        for (int i = 0; i < length; i++) {
            while (!second.isEmpty() && nums[second.peek()] < nums[i]) {
                res[second.pop()] = nums[i];
            }
            while (!stack.isEmpty() && nums[stack.peek()] < nums[i]) {
                temp.push(stack.pop());
            }
            while (!temp.isEmpty()){
                second.push(temp.pop());
            }
            stack.push(i);
        }
        return res;
    }
}

                          # 735. 行星碰撞open in new window

                          给定一个整数数组 asteroids，表示在同一行的行星。

                          对于数组中的每一个元素，其绝对值表示行星的大小，正负表示行星的移动方向（正表示向右移动，负表示向左移动）。每一颗行星以相同的速度移动。

                          找出碰撞后剩下的所有行星。碰撞规则：两个行星相互碰撞，较小的行星会爆炸。如果两颗行星大小相同，则两颗行星都会爆炸。两颗移动方向相同的行星，永远不会发生碰撞。

                          示例 1：

                          输入：asteroids = [5,10,-5]
输出：[5,10]
解释：10 和 -5 碰撞后只剩下 10 。 5 和 10 永远不会发生碰撞。

                          示例 2：

                          输入：asteroids = [8,-8]
输出：[]
解释：8 和 -8 碰撞后，两者都发生爆炸。

                          示例 3：

                          输入：asteroids = [10,2,-5]
输出：[10]
解释：2 和 -5 发生碰撞后剩下 -5 。10 和 -5 发生碰撞后剩下 10 。

                          提示：

                          • 2 <= asteroids.length <= 104
                          • -1000 <= asteroids[i] <= 1000
                          • asteroids[i] != 0

                          思路：

                          ​ 单调栈保存行星，只有栈顶是正数(向右)且下一颗是负数(向左)才会碰撞。

                          class Solution {
    public int[] asteroidCollision(int[] asteroids) {
        Deque<Integer> stack = new ArrayDeque<>();
        p:for (int asteroid : asteroids) {
            while (!stack.isEmpty() && stack.peek() > 0 && asteroid < 0) {
                if (stack.peek() < Math.abs(asteroid)) {
                    stack.pop();
                } else if (stack.peek() == Math.abs(asteroid)) {
                    stack.pop();
                    continue p;
                } else {
                    continue p;
                }
            }
            stack.push(asteroid);
        }
        int[] res = new int[stack.size()];
        for (int i = 0; i < res.length; i++) {
            res[i] = stack.pollLast();
        }
        return res;
    }
}

                          # 1019. 链表中的下一个更大节点open in new window

                          给定一个长度为 n 的链表 head

                          对于列表中的每个节点，查找下一个 更大节点 的值。也就是说，对于每个节点，找到它旁边的第一个节点的值，这个节点的值 严格大于 它的值。

                          返回一个整数数组 answer ，其中 answer[i] 是第 i 个节点( 从1开始 )的下一个更大的节点的值。如果第 i 个节点没有下一个更大的节点，设置 answer[i] = 0 。

                          示例 1：

                          输入：head = [2,1,5]
输出：[5,5,0]

                          示例 2：

                          输入：head = [2,7,4,3,5]
输出：[7,0,5,5,0]

                          提示：

                          • 链表中节点数为 n
                          • 1 <= n <= 10^4
                          • 1 <= Node.val <= 10^9

                          思路：

                          ​ 简单的单调栈

                          class Solution {
    public int[] nextLargerNodes(ListNode head) {
        Deque<Map.Entry<Integer, Integer>> stack = new ArrayDeque<>();
        ListNode temp = head;
        int length = 0;
        for (; temp != null; length++, temp = temp.next) ;
        int[] res = new int[length];
        length = 0;
        while (head != null) {
            while (!stack.isEmpty() && stack.peek().getValue() < head.val) {
                Map.Entry<Integer, Integer> pop = stack.pop();
                res[pop.getKey()] = head.val;
            }
            stack.push(Map.entry(length, head.val));
            length++;
            head = head.next;
        }
        return res;
    }
}

                          # 1475. 商品折扣后的最终价格open in new window

                          给你一个数组 prices ，其中 prices[i] 是商店里第 i 件商品的价格。

                          商店里正在进行促销活动，如果你要买第 i 件商品，那么你可以得到与 prices[j] 相等的折扣，其中 j 是满足 j > i 且 prices[j] <= prices[i] 的 最小下标 ，如果没有满足条件的 j ，你将没有任何折扣。

                          请你返回一个数组，数组中第 i 个元素是折扣后你购买商品 i 最终需要支付的价格。

                          示例 1：

                          输入：prices = [8,4,6,2,3]
输出：[4,2,4,2,3]
解释：
商品 0 的价格为 price[0]=8 ，你将得到 prices[1]=4 的折扣，所以最终价格为 8 - 4 = 4 。
商品 1 的价格为 price[1]=4 ，你将得到 prices[3]=2 的折扣，所以最终价格为 4 - 2 = 2 。
商品 2 的价格为 price[2]=6 ，你将得到 prices[3]=2 的折扣，所以最终价格为 6 - 2 = 4 。
商品 3 和 4 都没有折扣。

                          示例 2：

                          输入：prices = [1,2,3,4,5]
输出：[1,2,3,4,5]
解释：在这个例子中，所有商品都没有折扣。

                          示例 3：

                          输入：prices = [10,1,1,6]
输出：[9,0,1,6]

                          提示：

                          • 1 <= prices.length <= 500
                          • 1 <= prices[i] <= 10^3

                          思路：

                          ​ 维护一个单调递增栈，当当前价格小于等于栈顶元素价格时，进行相应的折扣。

                          class Solution {
    public int[] finalPrices(int[] prices) {
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < prices.length; i++) {
            while (!stack.isEmpty() && prices[i] <= prices[stack.peek()]) {
                int index = stack.pop();
                prices[index] = prices[index] - prices[i];
            }
            stack.push(i);
        }
        return prices;
    }
}

                          # 901. 股票价格跨度open in new window

                          设计一个算法收集某些股票的每日报价，并返回该股票当日价格的 跨度 。

                          当日股票价格的 跨度 被定义为股票价格小于或等于今天价格的最大连续日数（从今天开始往回数，包括今天）。

                          • 例如，如果未来 7 天股票的价格是 [100,80,60,70,60,75,85]，那么股票跨度将是 [1,1,1,2,1,4,6] 。

                          实现 StockSpanner 类：

                          • StockSpanner() 初始化类对象。
                          • int next(int price) 给出今天的股价 price ，返回该股票当日价格的 跨度 。

                          示例：

                          输入：
["StockSpanner", "next", "next", "next", "next", "next", "next", "next"]
[[], [100], [80], [60], [70], [60], [75], [85]]
输出：
[null, 1, 1, 1, 2, 1, 4, 6]

解释：
StockSpanner stockSpanner = new StockSpanner();
stockSpanner.next(100); // 返回 1
stockSpanner.next(80);  // 返回 1
stockSpanner.next(60);  // 返回 1
stockSpanner.next(70);  // 返回 2
stockSpanner.next(60);  // 返回 1
stockSpanner.next(75);  // 返回 4 ，因为截至今天的最后 4 个股价 (包括今天的股价 75) 都小于或等于今天的股价。
stockSpanner.next(85);  // 返回 6

                          提示：

                          • 1 <= price <= 10^5
                          • 最多调用 next 方法 10^4 次

                          思路：

                          ​ 使用单调栈保存价格和输入的日期，题目的要求等价于找到上一个更大的数，然后计算两个下标的差值，对于小于当前价格的值，也不可能做为上一个更大的数，对无用数据进行移除。

                          class StockSpanner {
    Deque<Map.Entry<Integer, Integer>> stack = new ArrayDeque<>(List.of(Map.entry(-1, Integer.MAX_VALUE)));
    int cur = 0;

    public int next(int price) {
        while (!stack.isEmpty() && stack.peek().getValue() <= price) {
            stack.pop();
        }
        int res = cur - stack.peek().getKey();
        stack.push(Map.entry(cur++, price));
        return res;
    }
}

                          思路：

                          ​ 使用哈希表保存每次价格出现时的连续天数，然后更新哈希表。

                          class StockSpanner {
    Deque<Integer> stack = new ArrayDeque<>(List.of(Integer.MAX_VALUE));
    Map<Integer, Integer> map = new HashMap<>();

    public int next(int price) {
        int res = 1;
        for (; price >= stack.peek(); res += map.get(stack.pop())) ;

        stack.push(price);

    public long maximumSumOfHeights(List<Integer> maxHeights) {
        int length = maxHeights.size();
        Deque<Integer> stack = new ArrayDeque<>(List.of(length));
        long[] prefix = new long[length], suffix = new long[length + 1];
        long sum = 0;
        for (int i = length - 1; i > -1; i--) {
            while (stack.size() > 1 && maxHeights.get(stack.peek()) >= maxHeights.get(i)) {
                int j = stack.pop();
                sum -= (long) maxHeights.get(j) * (stack.peek() - j);
            }
            sum += (long) maxHeights.get(i) * (stack.peek() - i);
            suffix[i] = sum;
            stack.push(i);
        }
        stack = new ArrayDeque<>(List.of(-1));
        sum = 0;
        long res = sum;
        for (int i = 0; i < length; i++) {
            while (stack.size() > 1 && maxHeights.get(stack.peek()) >= maxHeights.get(i)) {
                int j = stack.pop();
                sum -= (long) maxHeights.get(j) * (j - stack.peek());
            }
            sum += (long) maxHeights.get(i) * (i - stack.peek());
            prefix[i] = sum;
            res = Math.max(res, prefix[i] + suffix[i + 1]);
            stack.push(i);
        }
        return res;

                          ​ 类似962. 最大宽度坡open in new window，但本题要求是求最短的坡，观察坡的特点可以得到，坡的右边界左边的数一定比它小，否则我们取左边的数得到的坡更短，所以右边界可以构成一个单调非递增的栈。另外本题要求连续，所以栈中的元素必须是连续的，如果不满足则直接退出遍历。遇到相同的元素取更靠左的那个，可以避免重复的计算，然后如果栈顶已经是0了，说明整个数组是非递减的。然后遍历左边界，如果左边界非递减，则贪心向右继续遍历，如果大于栈顶，将栈顶出栈，继续进行遍历。栈中可以添加一个哨兵节点，方便计算移除的元素个数。

    public int findLengthOfShortestSubarray(int[] arr) {
        int length = arr.length, res;
        Deque<Integer> stack = new ArrayDeque<>(List.of(length - 1, length));
        for (int i = length - 2; i >= 0; i--) {
            if (arr[stack.peek()] > arr[i]) {
                stack.push(i);
            } else if (arr[stack.peek()] == arr[i]) {
                stack.pop();
                stack.push(i);
            } else {
                break;
            }
        }
        if (stack.peek() == 0) return 0;
        res = stack.peek();
        for (int i = 0; i < arr.length; i++) {
            if (i > 0 && arr[i] < arr[i - 1]) break;
            while (stack.size() > 1 && arr[stack.peek()] < arr[i]) {
                stack.pop();
            }
            res = Math.min(res, stack.peek() - i - 1);
        }
        return res;
    }
}

                          # 42. 接雨水open in new window

                          给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

                          示例 1：

                          输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

                          示例 2：

                          输入：height = [4,2,0,3,2,5]
输出：9

                          思路：

                          ​ 找上一个最大的元素，在找的过程中填坑

                          ​ 单调递减栈保存下标，如果遇到高的点，说明可以乘住水，将栈顶出栈，栈顶下一个元素与当前的下标构成左右两边界，出栈的元素计算以它为底接的雨水。

                          class Solution {
    public int trap(int[] height) {
        int length = height.length, res = 0;
        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < length; i++) {
            while (!stack.isEmpty() && height[stack.peek()] < height[i]) {
                int cur = stack.pop();
                if (stack.isEmpty()) {
                    break;
                }
                int pre = stack.peek();
                // 相当于开区间(peek, i)之间的区域能接雨水，所以计算底的长度为(i - peek - 1)
                int width = i - pre - 1, high = Math.min(height[pre], height[i]) - height[cur];
                res += width * high;
            }
            stack.push(i);
        }
        return res;
    }
}

                          给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

                          输入： heights = [2,4]

                          思路：

                          ​ 想象成锯木板，如果木板都是递增的那我很开心，如果突然遇到一块木板i矮了一截，那我就先找之前最戳出来的一块（其实就是第i-1块），计算一下这个木板单独的面积，然后把它锯成次高的，这是因为我之后的计算都再也用不着这块木板本身的高度了。再然后如果发觉次高的仍然比现在这个i木板高，那我继续单独计算这个次高木板的面积（应该是第i-1和i-2块），再把它俩锯短。直到发觉不需要锯就比第i块矮了，那我继续开开心心往右找更高的。当然为了避免到了最后一直都是递增的，所以可以在最后加一块高度为0的木板。前面可以也加一块，方便计算。

                          class Solution {
    public int largestRectangleArea(int[] heights) {
        int res = 0;
        Deque<Integer> stack = new ArrayDeque<>(List.of(0));
        for (int i = 0; i <= heights.length; i++) {
            int num = i == heights.length ? 0 : heights[i];
            while (!stack.isEmpty() && num < (stack.peek() == 0 ? 0 : heights[stack.peek() - 1])) {
                int high = heights[stack.pop() - 1];
                res = Math.max(res, (i - stack.peek()) * high);
            }
            stack.push(i + 1);
        }
        return res;
    }
}

                          class Solution {
    public int minimumVisitedCells(int[][] grid) {
        int m = grid.length, n = grid[0].length, mn = 0;
        List<int[]>[] colSt = new ArrayList[n];
        Arrays.setAll(colSt, e -> new ArrayList<int[]>());
        for (int i = m - 1; i >= 0; --i) {
            List<int[]> stack = new ArrayList<>();
            for (int j = n - 1; j >= 0; --j) {
                List<int[]> stack2 = colSt[j];
                mn = Integer.MAX_VALUE;
                int g = grid[i][j];
                if (i == m - 1 && j == n - 1) {
                    mn = 0;
                } else if (g > 0) {
                    int k = search(stack, j + g);
                    if (k < stack.size()) {
                        mn = Math.min(mn, stack.get(k)[0]);
                    }
                    k = search(stack2, i + g);
                    if (k < stack2.size()) {
                        mn = Math.min(mn, stack2.get(k)[0]);
                    }
                }
                if (mn == Integer.MAX_VALUE) {
                    continue;
                }
                ++mn;
                while (!stack.isEmpty() && mn <= stack.get(stack.size() - 1)[0]) {
                    stack.remove(stack.size() - 1);
                }
                stack.add(new int[]{mn, j});
                while (!stack2.isEmpty() && mn <= stack2.get(stack2.size() - 1)[0]) {
                    stack2.remove(stack2.size() - 1);
                }
                stack2.add(new int[]{mn, i});
            }
        }
        return mn < Integer.MAX_VALUE ? mn : -1;
    }

    private int search(List<int[]> st, int target) {
        int left = -1, right = st.size();
        while (left + 1 < right) {
            int mid = (left + right) >>> 1;
            if (st.get(mid)[1] > target) {
                left = mid;
            } else {
                right = mid;
            }
        }
        return right;
    }
}

            deque.pollLast();
        }
        deque.offerLast(value);
    }

                          ​ 子数组的和可以转化为前缀和的差，所以可以用一个单调递增队列保存最小的前缀和使得差值能尽可能满足k，枚举所有前缀和，使得差值满足k，如果满足则贪心移除队列头，判断使用队列第二小的前缀和能否满足条件(如果队列头当前可以满足，之后的遍历也不会再取了，因为长度当前更短)。维护队列时，注意等号，即值相等时，优先取右边的前缀和使得长度更小。

                          class Solution {
    public int longestSubarray(int[] nums, int limit) {
        Deque<Integer> max = new ArrayDeque<>();
        Deque<Integer> min = new ArrayDeque<>();
        int left = 0, res = 0;
        for (int right = 0; right < nums.length; right++) {
            while (!max.isEmpty() && nums[right] > max.peekLast()) {
                max.pollLast();
            }
            while (!min.isEmpty() && nums[right] < min.peekLast()) {
                min.pollLast();
            }
            max.offerLast(nums[right]);
            min.offerLast(nums[right]);
            while (max.peekFirst() - min.peekFirst() > limit) {
                if (max.peekFirst() == nums[left]) max.pollFirst();
                if (min.peekFirst() == nums[left]) min.pollFirst();
                left++;
            }
            res = Math.max(res, right - left + 1);
        }
        return res;
    }
}
