    # 2567.修改两个元素的最小分数open in new window

    给你一个下标从 0 开始的整数数组 nums 。

    • nums 的 最小 得分是满足 0 <= i < j < nums.length 的 |nums[i] - nums[j]| 的最小值。
    • nums的 最大 得分是满足 0 <= i < j < nums.length 的 |nums[i] - nums[j]| 的最大值。
    • nums 的分数是 最大 得分与 最小 得分的和。

    我们的目标是最小化 nums 的分数。你 最多 可以修改 nums 中 2 个元素的值。

    请你返回修改 nums 中 至多两个 元素的值后，可以得到的 最小分数 。

    |x| 表示 x 的绝对值。

    示例 1：

    输入：nums = [1,4,3]
输出：0
解释：将 nums[1] 和 nums[2] 的值改为 1 ，nums 变为 [1,1,1] 。|nums[i] - nums[j]| 的值永远为 0 ，所以我们返回 0 + 0 = 0 。

    示例 2：

    输入：nums = [1,4,7,8,5]
输出：3
解释：
将 nums[0] 和 nums[1] 的值变为 6 ，nums 变为 [6,6,7,8,5] 。
最小得分是 i = 0 且 j = 1 时得到的 |nums[i] - nums[j]| = |6 - 6| = 0 。
最大得分是 i = 3 且 j = 4 时得到的 |nums[i] - nums[j]| = |8 - 5| = 3 。
最大得分与最小得分之和为 3 。这是最优答案。

    提示：

    • 3 <= nums.length <= 10^5
    • 1 <= nums[i] <= 10^9

    思路：

    ​ 把最大的两个变成第三大，或把最小的两个变成第三小，或把最小的变为第二小并把最大的变为第二大。

    class Solution {
    public int minimizeSum(int[] nums) {
        int length = nums.length;
        Arrays.sort(nums);
        int score1 = nums[length - 3] - nums[0];
        int score2 = nums[length - 1] - nums[2];
        int score3 = nums[length - 2] - nums[1];
        return Math.min(score3, Math.min(score2, score1));
    }
}

    # 2383. 赢得比赛需要的最少训练时长open in new window

    你正在参加一场比赛，给你两个 正 整数 initialEnergy 和 initialExperience 分别表示你的初始精力和初始经验。

    另给你两个下标从 0 开始的整数数组 energy 和 experience，长度均为 n 。

    你将会 依次 对上 n 个对手。第 i 个对手的精力和经验分别用 energy[i] 和 experience[i] 表示。当你对上对手时，需要在经验和精力上都 严格 超过对手才能击败他们，然后在可能的情况下继续对上下一个对手。

    击败第 i 个对手会使你的经验 增加 experience[i]，但会将你的精力 减少 energy[i] 。

    在开始比赛前，你可以训练几个小时。每训练一个小时，你可以选择将增加经验增加 1 或者 将精力增加 1 。

    返回击败全部 n 个对手需要训练的 最少 小时数目。

    示例 1：

    输入：initialEnergy = 5, initialExperience = 3, energy = [1,4,3,2], experience = [2,6,3,1]
输出：8
解释：在 6 小时训练后，你可以将精力提高到 11 ，并且再训练 2 个小时将经验提高到 5 。
按以下顺序与对手比赛：
- 你的精力与经验都超过第 0 个对手，所以获胜。
  精力变为：11 - 1 = 10 ，经验变为：5 + 2 = 7 。
- 你的精力与经验都超过第 1 个对手，所以获胜。
  精力变为：10 - 4 = 6 ，经验变为：7 + 6 = 13 。
- 你的精力与经验都超过第 2 个对手，所以获胜。
  精力变为：6 - 3 = 3 ，经验变为：13 + 3 = 16 。
- 你的精力与经验都超过第 3 个对手，所以获胜。
  精力变为：3 - 2 = 1 ，经验变为：16 + 1 = 17 。
在比赛前进行了 8 小时训练，所以返回 8 。
可以证明不存在更小的答案。

    示例 2：

    输入：initialEnergy = 2, initialExperience = 4, energy = [1], experience = [3]
输出：0
解释：你不需要额外的精力和经验就可以赢得比赛，所以返回 0 。

    提示：

    • n == energy.length == experience.length
    • 1 <= n <= 100
    • 1 <= initialEnergy, initialExperience, energy[i], experience[i] <= 100

    # 2337. 移动片段得到字符串open in new window

    给你两个字符串 start 和 target ，长度均为 n 。每个字符串 仅 由字符 'L'、'R' 和 '_' 组成，其中：

    • 字符 'L' 和 'R' 表示片段，其中片段 'L' 只有在其左侧直接存在一个 空位 时才能向 左 移动，而片段 'R' 只有在其右侧直接存在一个 空位 时才能向 右 移动。
    • 字符 '_' 表示可以被 任意 'L' 或 'R' 片段占据的空位。

    如果在移动字符串 start 中的片段任意次之后可以得到字符串 target ，返回 true ；否则，返回 false 。

    示例 1：

    输入：start = "_L__R__R_", target = "L______RR"
输出：true
解释：可以从字符串 start 获得 target ，需要进行下面的移动：
- 将第一个片段向左移动一步，字符串现在变为 "L___R__R_" 。
- 将最后一个片段向右移动一步，字符串现在变为 "L___R___R" 。
- 将第二个片段向右移动散步，字符串现在变为 "L______RR" 。
可以从字符串 start 得到 target ，所以返回 true 。

    示例 2：

    输入：start = "R_L_", target = "__LR"
输出：false
解释：字符串 start 中的 'R' 片段可以向右移动一步得到 "_RL_" 。
但是，在这一步之后，不存在可以移动的片段，所以无法从字符串 start 得到 target 。

    示例 3：

    输入：start = "_R", target = "R_"
输出：false
解释：字符串 start 中的片段只能向右移动，所以无法从字符串 start 得到 target 。

    提示：

    • n == start.length == target.length
    • 1 <= n <= 10^5
    • start 和 target 由字符 'L'、'R' 和 '_' 组成

    一次跳跃的 长度 是青蛙跳跃前和跳跃后所在两块石头之间的距离。

    • 更正式的，如果青蛙从 stones[i] 跳到 stones[j] ，跳跃的长度为 |stones[i] - stones[j]| 。

    一条路径的 代价 是这条路径里的 最大跳跃长度 。

    示例 1：

    示例 3：

    # 13. 最大的数

    给定一个数 n，如 23121；给定一组数字 A 如 {2,4,9}，求由 A 中元素组成的、小于 n 的最大数。如小于 23121 的最大数为 22999。

    示例1：

    输入：n = 23121 , A = {2,4,9}
输出：22999

    示例2：

    输入：n = 2321 , A = {2,3,4}
输出：2244

    示例3：

    输入：n = 1229 , A = {1,9}
输出：1199

输入：n = 2229 , A = {1,9}
输出：1999

输入：n = 444359 , A = {4,5,9}
输出：99999

    思路：

    ​ 数位DP，本质是力扣902的变形，原题求个数，本题求最大，注意要求不能等于数字，则将原数字-1即可。利用数位DP思想和贪心的做法。有两处可以贪心，1，先获取n的长度-1位的最大数作为保底。然后递归，一旦当前位不是isLimit，则后面位都可以取最大。如果一直可以isLimit，则继续递归下去。

    class Solution {
    char[] s;
    int[] digits;
    int max;

    public int atMostNGivenDigitSet(int[] digits, int n) {
        this.digits = Arrays.stream(digits).sorted().toArray();
        this.s = String.valueOf(n - 1).toCharArray();
        for (int i = 0; i < s.length - 1; i++) {
            max = max * 10 + this.digits[digits.length - 1];
        }
        dfs(0, 0);
        return max == 0 ? -1 : max;
    }

    void dfs(int i, int mask) {
        if (i == s.length) {
            max = Math.max(max, mask);
            return;
        }

        int up = s[i] - '0';
        for (int digit : digits) {
            if (digit > up) {
                break;
            } else if (digit < up) {
                max = Math.max(max, makeMax(i, digit, mask));
            } else {
                dfs(i + 1, mask * 10 + digit);
            }
        }
    }

    int makeMax(int i, int digit, int mask) {
        mask = mask * 10 + digit;
        for (int j = i + 1; j < s.length; j++) {
            mask = mask * 10 + digits[digits.length - 1];
        }
        return mask;
    }

    public static void main(String[] args) {
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{2, 4, 9}, 23121)); // 22999
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{2, 3, 4}, 2321));  // 2244
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{1, 9}, 1229));     // 1199
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{1, 9}, 2229));     // 1999
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{4, 5, 9}, 444359));// 99999
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{4, 5, 9}, 444559));// 444555
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{2, 4, 8}, 248429));// 248428
        System.out.println(new Solution().atMostNGivenDigitSet(new int[]{2}, 1));           // -1
    }
}

    思路：

    ​ 回溯法，从高位到低位一次匹配，如果当前数在数组内，则优先取当前数，如果不在，则分情况。1、数组中存在更小的数，则使用这个数，然后后面的数直接取数组中最大的数。2、数组中不存在更小的数，则进行回溯，找到上一位有更小数的位数，然后将它变为更小的数，之后的位则使用最大的数。

    ​ 实现上，有两点可以注意，1、如果找不到更小的数，可以直接使用当前拼接的数再跑一次dfs，此时拿到的就是回溯后上一位有更小数的结果。2、如果第一位就找不到，则此时需要空跳，防止位数超标。

    class Solution {
    private int max = 0;
    private final TreeSet<Character> set = new TreeSet<>();

    public int maxNum(int[] a, int x) {
        for (int i : a) {
            set.add(Character.forDigit(i, 10));
            max = Math.max(max, i);
        }
        return dfs(x - 1);
    }

    int dfs(int x) {
        if (x <= 0) return 0;
        int res = 0;
        boolean lower = false;
        char[] chars = (x + "").toCharArray();
        for (int i = 0; i < chars.length; i++) {
            char c = chars[i];
            if (lower) {
                res = res * 10 + max;
                continue;
            }
            if (set.contains(c)) {
                res = res * 10 + (c - '0');
                continue;
            }
            if (set.lower(c) != null) {
                res = res * 10 + Character.digit(set.lower(c), 10);
            } else {
                res = i > 0 ? dfs(res - 1) * 10 + max : 0;
            }
            lower = true;
        }
        return res;
    }
}

    思路：

    ​ 对数组A排序，数字从小到大，可以先使用n位最小数组成的数与当前数比较，如果大的话，说明需要使用n - 1个最大数去构造。否则从高位到低位去匹配，优先使用当前数去填充一位，如果当前位不在数组中，则分情况讨论。1、有更小的数，则当前位使用更小的数，然后后面位直接使用最大数。2、没有更小的数，则从当前位开始全部使用最大数，回溯找到上一位有更小数的位，将其替换为更小数。

    public class Solution {
    public static int maxNum(int[] nums, int target) {
        StringBuilder res = new StringBuilder();
        target = target - 1;
        String str1 = target + "";
        int len = str1.length();
        TreeSet<Character> set = new TreeSet<>();
        for (int num : nums) {
            set.add(Character.forDigit(num, 10));
        }
        String str2 = String.valueOf(set.first()).repeat(len);
        int cases = str2.compareTo(str1);
        if (cases >= 0) {
            res = new StringBuilder(String.valueOf(set.last()).repeat(len - 1));
        } else {
            for (int i = 0; i < len; i++) {
                char cur = str1.charAt(i);
                if (set.contains(cur)) {
                    res.append(cur);
                    continue;
                }
                Character lower = set.lower(cur);
                if (lower != null) {
                    res.append(lower);
                    res.append((set.last() + "").repeat(len - i - 1));
                } else {
                    String last = (set.last() + "").repeat(len - i);
                    int j = i - 1;
                    for (; set.lower(res.charAt(j)) == null; j--) {
                        res.setCharAt(j, set.last());
                    }
                    res.setCharAt(j, set.lower(res.charAt(j)));
                    res.append(last);
                }
                break;
            }
        }
        return Integer.parseInt(res.toString());
    }
}
