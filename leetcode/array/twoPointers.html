                      # 双指针

                      [toc]

                      ​ 本质是枚举左右端点，通过两个指针找到两侧端点满足条件的部分，从而减少时间复杂度。具体的相向双指针，一般会用来解决两个端点都能满足条件，取更优的一方，另一方继续用来比较。

                      # 167. 两数之和 II - 输入有序数组open in new window

                      给你一个下标从 1 开始的整数数组 numbers ，该数组已按 非递减顺序排列 ，请你从数组中找出满足相加之和等于目标数 target 的两个数。如果设这两个数分别是 numbers[index1] 和 numbers[index2] ，则 1 <= index1 < index2 <= numbers.length 。

                      以长度为 2 的整数数组 [index1, index2] 的形式返回这两个整数的下标 index1 和 index2。

                      你可以假设每个输入 只对应唯一的答案 ，而且你 不可以 重复使用相同的元素。

                      你所设计的解决方案必须只使用常量级的额外空间。

                      示例 1：

                      输入：numbers = [2,7,11,15], target = 9
输出：[1,2]
解释：2 与 7 之和等于目标数 9 。因此 index1 = 1, index2 = 2 。返回 [1, 2] 。

                      示例 2：

                      输入：numbers = [2,3,4], target = 6
输出：[1,3]
解释：2 与 4 之和等于目标数 6 。因此 index1 = 1, index2 = 3 。返回 [1, 3] 。

                      示例 3：

                      输入：numbers = [-1,0], target = -1
输出：[1,2]
解释：-1 与 0 之和等于目标数 -1 。因此 index1 = 1, index2 = 2 。返回 [1, 2] 。

                      提示：

                      • 2 <= numbers.length <= 3 * 10^4
                      • -1000 <= numbers[i] <= 1000
                      • numbers 按 非递减顺序 排列
                      • -1000 <= target <= 1000
                      • 仅存在一个有效答案

                      思路：

                      ​ 双指针分别指向最小和最大的数，然后判断与target的大小关系。如果不满足要求，可以从头或尾排除一个元素，因为数组是有序的，如果最小+最大都不能满足，则次小+最大更不能满足，此时排除一个数，并排除了n个数对，以此降低时间复杂度。

                      class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int length = numbers.length, left = 0, right = length - 1;
        while(left < right){
            int sum = numbers[left] + numbers[right];
            if(sum == target) return new int[]{left + 1, right + 1};
            else if(sum > target) right--;
            else left++;
        }
        return new int[0];
    }
}

                      # 15. 三数之和open in new window

                      给你一个包含n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

                      注意: 答案中不可以包含重复的三元组。

                      示例1：

                      输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]

                      示例2：

                      输入：nums = []
输出：[]

                      示例3：

                      输入：nums = [0]
输出：[]

                      思路：

                      ​ 先将数组排序。双指针，循环i作为第一个数，左指针指向i + 1，右指针指向数组末尾。判断三数之和，如果等于0，添加三元组，因为不能包含重复的元组，将等于left和right的值都加到或减到下一个值。如果小于0，右指针左移，如果大于0，左指针右移。

                      class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        List<List<Integer>> res = new ArrayList<>();
        for (int i = 0; i < n - 2; i++) {
            int x = nums[i];
            if (i > 0 && x == nums[i - 1]) continue;
            if (x + nums[i + 1] + nums[i + 2] > 0) break;
            if (x + nums[n - 1] + nums[n - 2] < 0) continue;
            int left = i + 1, right = n - 1;
            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == -x) {
                    res.add(List.of(x, nums[left], nums[right]));
                    while (left < right && nums[left] == nums[++left]) ;
                    while (right > left && nums[right] == nums[--right]) ;
                } else if (sum > -x) {
                    while (right > left && nums[right] == nums[--right]) ;
                } else {
                    while (left < right && nums[left] == nums[++left]) ;
                }
            }
        }
        return res;
    }
}

                      # 16. 最接近的三数之和open in new window

                      给你一个长度为 n 的整数数组 nums 和 一个目标值 target。请你从 nums 中选出三个整数，使它们的和与 target 最接近。

                      返回这三个数的和。

                      假定每组输入只存在恰好一个解。

                      示例 1：

                      输入：nums = [-1,2,1,-4], target = 1
输出：2
解释：与 target 最接近的和是 2 (-1 + 2 + 1 = 2) 。

                      示例 2：

                      输入：nums = [0,0,0], target = 1
输出：0

                      提示：

                      • 3 <= nums.length <= 1000
                      • -1000 <= nums[i] <= 1000
                      • -10^4 <= target <= 10^4

                      思路：

                      ​ 双指针分别指向最小值和最大值，然后判断和与目标的绝对值，如果更小则可以更新差值与答案，直到遍历完成。

                      class Solution {
    public int threeSumClosest(int[] nums, int target) {
        Arrays.sort(nums);
        int min = Integer.MAX_VALUE, res = 0;
        for (int i = 0; i < nums.length - 2; i++) {
            int x = nums[i];
            if (i > 0 && x == nums[i - 1]) continue;
            int left = i + 1, right = nums.length - 1;
            while (left < right) {
                int sum = x + nums[left] + nums[right];
                if (min > Math.abs(sum - target)) {
                    min = Math.abs(sum - target);
                    res = sum;
                }
                if (sum == target) return target;
                else if (sum > target) right--;
                else left++;
            }
        }
        return res;
    }
}

                      # 18. 四数之和open in new window

                      给你一个由 n 个整数组成的数组 nums ，和一个目标值 target 。请你找出并返回满足下述全部条件且不重复的四元组 [nums[a], nums[b], nums[c], nums[d]] ：

                      • 0 <= a, b, c, d < n

                      • a、b、c 和 d 互不相同

                      • nums[a] + nums[b] + nums[c] + nums[d] == target

                        你可以按任意顺序返回答案 。

                      示例1：

                      输入：nums = [1,0,-1,0,-2,2], target = 0
输出：[[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]

                      示例2：

                      输入：nums = [2,2,2,2,2], target = 8
输出：[[2,2,2,2]]

                      思路：

                      ​ 在三数之和的基础上，再加一层循环。

                      class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        Arrays.sort(nums);
        int n = nums.length;
        List<List<Integer>> res = new ArrayList<>();
        for (int i = 0; i < n - 3; i++) {
            int x = nums[i];
            if (i > 0 && x == nums[i - 1]) continue;
            if ((long) x + nums[i + 1] + nums[i + 2] + nums[i + 3] > target) break;
            if ((long) x + nums[n - 1] + nums[n - 2] + nums[n - 3] < target) continue;
            for (int j = i + 1; j < n - 2; j++) {
                int y = nums[j], left = j + 1, right = n - 1;
                if (j > i + 1 && nums[j] == nums[j - 1]) continue;
                if ((long) y + nums[i + 1] + nums[i + 2] > target - x) break;
                if ((long) y + nums[n - 1] + nums[n - 2] < target - x) continue;
                while (left < right) {
                    long sum = nums[left] + nums[right];
                    if (sum == ((long) target - x - y)) {
                        res.add(List.of(x, y, nums[left], nums[right]));
                        while (left < right && nums[left] == nums[++left]) ;
                        while (right > left && nums[right] == nums[--right]) ;
                    } else if (sum > ((long) target - x - y)) {
                        while (right > left && nums[right] == nums[--right]) ;
                    } else {
                        while (left < right && nums[left] == nums[++left]) ;
                    }
                }
            }
        }
        return res;
    }
}

                      # 27. 移除元素open in new window

                      给你一个数组nums和一个值val，你需要原地移除所有数值等于val的元素，并返回移除后数组的新长度。

                      不要使用额外的数组空间，你必须仅使用 O (1) 额外空间并原地修改输入数组。

                      元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

                      示例1:

                      输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。

                      示例2:

                      输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3]
解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。

                      提示：

                      • 0 <= nums.length <= 100
                      • 0 <= nums[i] <= 50
                      • 0 <= val <= 100

                      思路：

                      ​ 快慢指针，快指针寻找不等于val的下标，慢指针寻找等于val的下标，更新快指针的值到慢指针下标处

                      class Solution {
    public int removeElement(int[] nums, int val) {
        int slow = 0;
        for (int fast = 0; fast < nums.length; fast++) {
            if (nums[fast] != val) {
                nums[slow++] = nums[fast];
            }
        }
        return slow;
    }
}

                      # 26. 删除有序数组中的重复项open in new window

                      给你一个有序数组nums ，请你 原地 删除重复出现的元素，使每个元素只出现一次 ，返回删除后数组的新长度。

                      不要使用额外的数组空间，你必须在原地修改输入数组 并在使用O (1) 额外空间的条件下完成。

                      示例1：

                      输入：nums = [1,1,2]
输出：2, nums = [1,2]
解释：函数应该返回新的长度 2 ，并且原数组 nums 的前两个元素被修改为 1, 2 。不需要考虑数组中超出新长度后面的元素。

                      示例2：

                      输入：nums = [0,0,1,1,1,2,2,3,3,4]
输出：5, nums = [0,1,2,3,4]
解释：函数应该返回新的长度 5 ， 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4 。不需要考虑数组中超出新长度后面的元素。

                      思路：

                      ​ 双指针，慢指针停留在每个元素第一次出现的地方后一个位置，因为有序，所以快指针寻找大于慢指针-1处不同元素，找到后在慢指针的下标处存放快指针指向的值.

                      class Solution {
    public int removeDuplicates(int[] nums) {
        int slow = 1;
        for (int fast = 1; fast < nums.length; fast++) {
            if (nums[fast] > nums[slow - 1]) {
                nums[slow++] = nums[fast];
            }
        }
        return slow;
    }
}

                      # 80. 删除有序数组中的重复项 IIopen in new window

                      给你一个有序数组 nums ，请你原地open in new window删除重复出现的元素，使每个元素 最多出现两次 ，返回删除后数组的新长度。

                      不要使用额外的数组空间，你必须在 原地open in new window修改输入数组 并在使用 O (1) 额外空间的条件下完成。

                      示例 1：

                      输入：nums = [1,1,1,2,2,3]
输出：5, nums = [1,1,2,2,3]
解释：函数应返回新长度 length = 5, 并且原数组的前五个元素被修改为 1, 1, 2, 2, 3 。 不需要考虑数组中超出新长度后面的元素。

                      示例 2：

                      输入：nums = [0,0,1,1,1,1,2,3,3]
输出：7, nums = [0,0,1,1,2,3,3]
解释：函数应返回新长度 length = 7, 并且原数组的前五个元素被修改为 0, 0, 1, 1, 2, 3, 3 。 不需要考虑数组中超出新长度后面的元素。

                      思路：

                      ​ 使用一个数记录第一次出现的值，当第二次出现时也可以填入，然后将临时数置为INF。

                      class Solution {
    public int removeDuplicates(int[] nums) {
        int temp = nums[0], slow = 1;
        for (int fast = 1; fast < nums.length; fast++) {
            if (nums[fast] > nums[slow - 1]) {
                nums[slow++] = nums[fast];
                temp = nums[fast];
            } else if (nums[fast] == nums[slow - 1] && temp == nums[fast]) {
                nums[slow++] = nums[fast];
                temp = -10001;
            }
        }
        return slow;
    }
}

                      思路：

                      ​ 记录一个下标位置，遍历数组，只有当下标小于2或者前两位的数不等于当前值(即不重复)，下标才可放置当前值。

                      class Solution {
    public int removeDuplicates(int[] nums) {
        int i = 0;
        for (int x : nums) {
            if (i < 2 || nums[i - 2] != x) {
                nums[i++] = x;
            }
        }
        return i;
    }
}

                      # 977. 有序数组的平方open in new window

                      给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

                      示例1：

                      输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]

                      示例2：

                      输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]

                      思路：

                      ​ 绝对值最大的在数组两侧，双指针指向两侧，比较两侧平方的大小，大的值放到最后，更新index

                      class Solution {
    public int[] sortedSquares(int[] nums) {
        nums = Arrays.stream(nums).map(i -> i * i).toArray();
        int length = nums.length, left = 0, right = length - 1;
        int[] res = new int[length];
        for (int i = length - 1; i > -1; i--) {
            res[i] = nums[left] > nums[right] ? nums[left++] : nums[right--];
        }
        return res;
    }
}

                      # 11. 盛最多水的容器open in new window

                      给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

                      找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

                      返回容器可以储存的最大水量。

                      说明：你不能倾斜容器。

                      示例 1：

                      输入：[1,8,6,2,5,4,8,3,7]
输出：49 
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

                      示例 2：

                      输入：height = [1,1]
输出：1

                      提示：

                      • n == height.length
                      • 2 <= n <= 10^5
                      • 0 <= height[i] <= 10^4

                      思路：

                      ​ 枚举，从左右端点开始选定两条线，计算面积，更新最大值，哪条边短则将哪条边移动。如果使用了两条边来圈定面积，根据木桶原理，水量决定于宽度和短边的高度。此时无论中间的线比短边长还是短，长则面积高不边，宽减少。短则高、宽都减少，不可能更大。所以直接把短边去掉，继续找剩余的线框定的面积。

                      class Solution {
    public int maxArea(int[] height) {
        int left = 0, right = height.length - 1, res = 0;
        while(left < right){
            int area = (right - left) * Math.min(height[left], height[right]);
            res = Math.max(res, area);
            if(height[left] < height[right]) left++;
            else right--;
        }
        return res;
    }
}

                      # 42. 接雨水open in new window

                      给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

                      示例 1：

                      输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

                      示例 2：

                      输入：height = [4,2,0,3,2,5]
输出：9

                      提示：

                      • n == height.length
                      • 1 <= n <= 2 * 10^4
                      • 0 <= height[i] <= 10^5

                      思路：

                      ​ 前后缀分解，分别得到从前面到当前点的最长边，和从后面到当前点的最长边，根据木桶原理，较短的那个就是木桶高度，再减去木桶自身的高度，则可得到当前这一个桶的乘水量。

                      class Solution {
    public int trap(int[] height) {
        int length = height.length, res = 0;
        int[] preMax = new int[length], sufMax = new int[length];
        preMax[0] = height[0];
        sufMax[length - 1] = height[length - 1];
        for(int i = 1; i < length; i++){
            preMax[i] = Math.max(preMax[i - 1], height[i]);
        }
        for(int i = length - 2; i > -1; i--){
            sufMax[i] = Math.max(sufMax[i + 1], height[i]);
        }
        for(int i = 0; i < length; i++){
            res += Math.min(preMax[i], sufMax[i]) - height[i];
        }
        return res;
    }
}

                      思路：

                      ​ 在上面思路上可进行优化，使用两个值滚动维护到当前位置时前缀和后缀的最大值，使用双指针进行维护，哪个值小则将指针进行移动。重点在于，如果i当前前缀的最大值小于当前后缀的最大值，那么易得它肯定小于i的后缀的最大值。可以使用i的前缀当作短板去计算当前装的雨水。后缀更小时同理

                      class Solution {
    public int trap(int[] height) {
        int length = height.length, res = 0, left = 0, right = length - 1, preMax = 0, sufMax = 0;
        // 等于，因为相等时也要计算当前桶装的水
        while (left <= right){
            preMax = Math.max(preMax, height[left]);
            sufMax = Math.max(sufMax, height[right]);
            if (preMax < sufMax) res += preMax - height[left++];
            else res += sufMax - height[right--];
        }
        return res;
    }
}

                      # 2565. 最少得分子序列open in new window

                      给你两个字符串 s 和 t 。

                      你可以从字符串 t 中删除任意数目的字符。

                      如果没有从字符串 t 中删除字符，那么得分为 0 ，否则：

                      • 令 left 为删除字符中的最小下标。
                      • 令 right 为删除字符中的最大下标。

                      字符串的得分为 right - left + 1 。

                      请你返回使 t 成为 s 子序列的最小得分。

                      一个字符串的 子序列 是从原字符串中删除一些字符后（也可以一个也不删除），剩余字符不改变顺序得到的字符串。（比方说 "ace" 是 "abcde" 的子序列，但是 "aec" 不是）。

                      示例 1：

                      输入：s = "abacaba", t = "bzaa"
输出：1
解释：这个例子中，我们删除下标 1 处的字符 "z" （下标从 0 开始）。
字符串 t 变为 "baa" ，它是字符串 "abacaba" 的子序列，得分为 1 - 1 + 1 = 1 。
1 是能得到的最小得分。

                      示例 2：

                      输入：s = "cde", t = "xyz"
输出：3
解释：这个例子中，我们将下标为 0， 1 和 2 处的字符 "x" ，"y" 和 "z" 删除（下标从 0 开始）。
字符串变成 "" ，它是字符串 "cde" 的子序列，得分为 2 - 0 + 1 = 3 。
3 是能得到的最小得分。

                      提示：

                      # NC142 最长重复子串open in new window

                      定义重复字符串是由两个相同的字符串首尾拼接而成。例如："abcabc" 是一个长度为 6 的重复字符串，因为它由两个 "abc" 串拼接而成；"abcba" 不是重复字符串，因为它不能由两个相同的字符串拼接而成。

                      给定一个字符串，请返回其最长重复子串的长度。

                      若不存在任何重复字符子串，则返回 0。

                      本题中子串的定义是字符串中一段连续的区间。

                      数据范围：字符串长度不大于 103103，保证字符串一定由小写字母构成。

                      进阶：空间复杂度 O(1)O(1)O(1)，时间复杂度 O(n2)O(n^2)O(n2)

                      示例1

                      输入：

                      "ababc"

                      返回值：

                      4

                      说明：

                      abab为最长的重复字符子串，长度为4

                      示例2

                      输入：

                      "abcab"

                      返回值：

                      0

                      说明：

                      该字符串没有重复字符子串

                      思路：

                      ​ 暴力法，会从最长的长度(n/2)开始，枚举每个长度，然后判断这个长度＊2的每个子串是否是重复的。使用双指针来优化这个过程，依然从最长的长度开始遍历，使用两个指针分别比较(i, j)。然后用一个数来保存相同的字符个数。遇到不同则置0，如果满足个数等于长度k，则找到了答案，相当于使用同向双指针。

                      
public class Solution {
    public int solve(String a) {
        int n = a.length(), count = 0;
        for (int i = n / 2; i > 0; i--) {
            for (int j = 0; j + i < n; j++) {
                if (a.charAt(j) == a.charAt(j + i)) {
                    count++;
                } else {
                    count = 0;
                }
                if (count == i) {
                    return i * 2;
                }
            }
        }
        return 0;
    }
}

                      # 581. 最短无序连续子数组open in new window

                      思路：

                      ​ 可以把数组分成三段，左段有序，中段无序，右端有序。

                      ​ 先只考虑中段数组，设其左边界为L，右边界为R：

                      • nums[R] 不可能是【L，R】中的最大值（否则应该将 nums[R] 并入右端数组）
                      • nums[L] 不可能是【L，R】中的最小值（否则应该将 nums[L] 并入左端数组）

                      很明显:

                      • 【L，R】中的最大值 等于【0，R】中的最大值，设其为 max
                      • 【L，R】中的最小值 等于 【L， nums.length-1】中的最小值，设其为 min

                      那么有：

                      • nums[R] < max < nums[R+1] < nums[R+2] < ... 所以说，从左往右遍历，最后一个小于max的为右边界
                      • nums[L] > min > nums[L-1] > nums[L-2] > ... 所以说，从右往左遍历，最后一个大于min的为左边界

                      最终用R - L + 1就是答案，极端情况下会缺失左段或右段数组，或者两段都缺失，这种情况也能囊括在内。

                      # 658. 找到 K 个最接近的元素open in new window

                      使用双指针，将更不靠近的一端指针进行移动，直到指针范围内的数据满足k的条件

                      class Solution {
    public List<Integer> findClosestElements(int[] arr, int k, int x) {
        int n = arr.length;
        int left = 0, right = n - 1;
        while (right - left + 1 > k) {
            if (Math.abs(arr[left] - x) > Math.abs(arr[right] - x)) {
                left++;
            } else {
                right--;
            }
        }
        return IntStream.rangeClosed(left, right).map(i -> arr[i]).boxed().toList();
    }
}

                      # 923. 三数之和的多种可能open in new window

                      ​ 存在重复元素下，使用双指针计算的题目，当左右指针相等时，使用组合数据公式计算C(n,2)，即n*(n-1)/2。不相等时，先计算范围内的贡献，再移动指针

                      class Solution {
    int[] nums;

    public int threeSumMulti(int[] arr, int target) {
        long res = 0, mod = (long) 1e9 + 7;
        Arrays.sort(arr);
        this.nums = arr;
        for (int i = 0; i < nums.length - 2; i++) {
            int left = i + 1, right = nums.length - 1, realTarget = target - nums[i];
            int x = twoSum(left, right, realTarget);
            res += x;
            res %= mod;
        }
        return (int) res;
    }

    int twoSum(int left, int right, int target) {
        int res = 0;
        while (left < right) {
            int x = nums[left] + nums[right];
            if (x == target) {
                if (nums[left] == nums[right]) {
                    res += (right - left + 1) * (right - left) / 2;
                    break;
                }
                int leftCount = 1, rightCount = 1;
                while (left < right && nums[left + 1] == nums[left]) {
                    left++;
                    leftCount++;
                }
                while (left < right && nums[right - 1] == nums[right]) {
                    right--;
                    rightCount++;
                }
                res += leftCount * rightCount;
                left++;
                right--;
            } else if (x < target) {
                left++;
            } else {
                right--;
            }
        }
        return res;
    }
}

                      # 划分问题

                      # 31 · 数组划分open in new window

                      对于partition问题，使用双指针来解决，注意判断左右指针时，用到了等号，主要是保证最终结束的时候right和left的位置是交错的，方便直接使用指针。

                      public class Solution {
    public int partitionArray(int[] nums, int k) {
        int n = nums.length;
        int left = 0, right = n - 1;
        while (left <= right) {
            while (left <= right && nums[left] < k) {
                left++;
            }
            while (left <= right && nums[right] >= k) {
                right--;
            }
            if (left <= right) {
                swap(nums, left, right);
            }
        }
        return left;
    }

    void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}

                      # 144 · 交错正负数open in new window

                      public class Solution {
    public void rerange(int[] a) {
        int pos = 0, n = a.length;
        for (int x : a) {
            if (x > 0) {
                pos++;
            } else {
                pos--;
            }
        }
        if (pos == 0) {
            pos++;
        }
        int left = 0, right = n - 1;
        while (left <= right) {
            while (left <= right && a[left] * pos > 0) {
                left++;
            }
            while (left <= right && a[right] * pos < 0) {
                right--;
            }
            if (left <= right) {
                swap(a, left, right);
                left++;
                right--;
            }
        }
        int i = 1, j = left;
        while (i < j && j < a.length) {
            swap(a, i, j);
            i += 2;
            j += 1;
        }
    }

    void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}

                      # 392. 判断子序列open in new window

                      ​ 如果 s 不是空串，我们可以遍历 t，在遍历的过程中看能否匹配 s 的每个字母。按照子序列的定义，在遍历 t 的过程中，把没匹配到的字母删除，剩下的就是匹配的字母，即字符串 s，这就说明 s 是 t 的子序列。

                      class Solution {
    public boolean isSubsequence(String s, String t) {
        if (s.isEmpty()) {
            return true;
        }
        int i = 0;
        for (char c : t.toCharArray()) {
            if (s.charAt(i) == c && ++i == s.length()) { // 所有字符匹配完毕
                return true; // s 是 t 的子序列
            }
        }
        return false;
    }
}
