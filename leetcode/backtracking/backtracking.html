                      # 17. 电话号码的字母组合open in new window

                      给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。答案可以按任意顺序返回。

                      给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

                      示例 1：

                      输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]

                      示例 2：

                      输入：digits = ""
输出：[]

                      示例 3：

                      输入：digits = "2"
输出：["a","b","c"]

                      思路：

                      ​ 用一个path数组记录路径上的字母

                      ​ 当前操作：枚举path[i]要填入的字母

                      ​ 子问题：构造字符串>= i的部分

                      ​ 下一个子问题：构造字符串>= i + 1的部分

                      class Solution {
    String[] map = new String[]{"", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};
    char[] digits, path;
    List<String> res = new ArrayList<>();

    public List<String> letterCombinations(String digits) {
        int n = digits.length();
        if (n == 0) {
            return res;
        }
        this.digits = digits.toCharArray();
        path = new char[n];
        dfs(0);
        return res;
    }

    void dfs(int i) {
        if (i == path.length) {
            res.add(new String(path));
            return;
        }
        for (char digit : map[digits[i] - '0'].toCharArray()) {
            path[i] = digit;
            dfs(i + 1);
        }
    }
}

                      # 78. 子集open in new window

                      给你一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集（幂集）。

                      解集 不能 包含重复的子集。你可以按 任意顺序 返回解集。

                      示例 1：

                      输入：nums = [1,2,3]
输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

                      示例 2：

                      输入：nums = [0]
输出：[[],[0]]

                      思路：

                      ​ 输入的视角，每个数字选或不选，如果选的话，递归完后需要恢复现场。

                      ​ 当前操作：枚举第i个数字选或不选

                      ​ 子问题：从下标>= i的数字中构造子集

                      ​ 下一个子问题：从下标>= i + 1的数字中构造子集

                      class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    int[] nums;

    public List<List<Integer>> subsets(int[] nums) {
        this.nums = nums;
        dfs(0);
        return res;
    }

    void dfs(int i) {
        if (i == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }
        dfs(i + 1);
        path.add(nums[i]);
        dfs(i + 1);
        path.remove(path.size() - 1);
    }
}

                      思路：

                      ​ 答案视角，因为子集不要求顺序，所以规定一个顺序，如严格递增的顺序

                      ​ 当前操作：枚举一个下标j >= i的数字，加入path（i是枚举的起点）

                      ​ 子问题：从下标>= i的数字中构造子集

                      ​ 下一个子问题：从下标>= j + 1的数字中构造子集

        return res;
    }

    void dfs(int i) {
        res.add(new ArrayList<>(path));

            path.remove(path.size() - 1);

    }

                      # 90. 子集 IIopen in new window

                      给你一个整数数组 nums ，其中可能包含重复元素，请你返回该数组所有可能的子集（幂集）。

                      解集不能包含重复的子集。返回的解集中，子集可以按任意顺序排列。

                      示例 1：

                      输入：nums = [1,2,2]
输出：[[],[1],[1,2],[1,2,2],[2],[2,2]]

                      示例 2：

                      输入：nums = [0]
输出：[[],[0]]

                      思路：

                      ​ 相比前一题，加了重复的条件，先对数组排序，然后可以使用两种方法来去重，选或不选，使用set去重。

                      class Solution {
    Set<List<Integer>> res = new HashSet<>();
    List<Integer> path = new ArrayList<>();
    int[] nums;

    public List<List<Integer>> subsetsWithDup(int[] nums) {
        Arrays.sort(nums);
        this.nums = nums;
        dfs(0);
        return new ArrayList<>(res);
    }

    void dfs(int i) {
        if (i == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }
        dfs(i + 1);
        path.add(nums[i]);
        dfs(i + 1);
        path.remove(path.size() - 1);
    }
}

                      思路：

                      ​ 枚举选哪个，因为数组是有序的，所以枚举时，如果和前一个值相同则跳过。额外添加了牛客要求的字典序排序逻辑。

                      class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> path = new ArrayList<>();
    int[] nums;

    public List<List<Integer>> subsetsWithDup(int[] nums) {
        Arrays.sort(nums);
        this.nums = nums;
        dfs(0);

        res.sort(((o1, o2) -> {
            int size = Math.min(o1.size(), o2.size());
            for (int i = 0; i < size; i++) {
                int comparison = Integer.compare(o1.get(i), o2.get(i));
                if (comparison != 0) {
                    return comparison;
                }
            }
            return Integer.compare(o1.size(), o2.size());
        }));
        return res;
    }

    void dfs(int i) {
        res.add(new ArrayList<>(path));
        for (int j = i; j < nums.length; j++) {
            if (j > i && nums[j] == nums[j - 1]) {
                continue;
            }
            path.add(nums[j]);
            dfs(j + 1);
            path.remove(path.size() - 1);
        }
    }
}

                      # 93. 复原 IP 地址open in new window

                      给定一个只包含数字的字符串，用以表示一个 IP 地址，返回所有可能从 s 获得的 有效 IP 地址 。你可以按任何顺序返回答案。

                      有效 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 '.' 分隔。

                      例如："0.1.2.201" 和 "192.168.1.1" 是 有效 IP 地址，但是 "0.011.255.245"、"192.168.1.312" 和 "192.168@1.1" 是 无效 IP 地址。

                      示例 1：

                      输入：s = "25525511135"
输出：["255.255.11.135","255.255.111.35"]

                      示例 2：

                      输入：s = "0000"
输出：["0.0.0.0"]

                      示例 3：

                      输入：s = "1111"
输出：["1.1.1.1"]

                      示例 4：

                      输入：s = "010010"
输出：["0.10.0.10","0.100.1.0"]

                      示例 5：

                      输入：s = "101023"
输出：["1.0.10.23","1.0.102.3","10.1.0.23","10.10.2.3","101.0.2.3"]

                      思路：

                      ​ 枚举每一个字符是否可以作为ip的一部分，使用两个数字作为记数，一个记录当前下标的位置，一个记录ip地址里出现了几段数字。因为最多到255，所以每次截取1到3个字符作为一段。

                      public class Solution {
    Set<String> set = new HashSet<>();
    ArrayList<String> res = new ArrayList<>();
    List<String> path = new ArrayList<>();
    String s;
    
    public List<String> restoreIpAddresses(String s) {
        this.s = s;
        for (int i = 0; i < 256; i++) {
            set.add(String.valueOf(i));
        }
        dfs(0, new StringBuilder());
        return res;
    }

    void dfs(int i, StringBuilder builder) {
        if (i == s.length()) {
            if (4 == path.size() && builder.isEmpty()) {
                res.add(String.join(".", path));
            }
            return;
        }
        if (path.size() > 4 || !set.contains(builder.toString() + s.charAt(i))) {
            return;
        }
        builder.append(s.charAt(i));
        path.add(builder.toString());
        dfs(i + 1, new StringBuilder());
        path.removeLast();
        dfs(i + 1, builder);
    }
}

                      给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是 回文串 。返回 s 所有可能的分割方案。

                      回文串 是正着读和反着读都一样的字符串。

                      示例 1：

                      输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]

                      示例 2：

                      输入：s = "a"
输出：[["a"]]

                      思路：

                      class Solution {

    List<String> path = new ArrayList<>();
    String s;

    public List<List<String>> partition(String s) {
        this.s = s;
        dfs(0, 0);
        return res;
    }

    void dfs(int i, int start) {
        if (i == s.length()) {
            res.add(new ArrayList<>(path));
            return;
        }
        if (i < s.length() - 1) {
            dfs(i + 1, start);
        }
        if (isPalindrome(start, i)) {

            dfs(i + 1, i + 1);
            path.remove(path.size() - 1);
        }

        while (left < right) {
            if (s.charAt(left++) != s.charAt(right--)) {
                return false;
            }
        }
        return true;
    }
}

                      # 139. 单词拆分open in new window

                      给定一个非空字符串 s 和一个包含非空单词的列表 wordDict，判定 s 是否可以被空格拆分为一个或多个在字典中出现的单词。

                      说明：

                      • 拆分时可以重复使用字典中的单词。
                      • 你可以假设字典中没有重复的单词。

                      示例 1：

                      输入: s = "leetcode", wordDict = ["leet", "code"]
输出: true
解释: 返回 true 因为 "leetcode" 可以被拆分成 "leet code"。

                      示例 2：

                      输入: s = "applepenapple", wordDict = ["apple", "pen"]
输出: true
解释: 返回 true 因为 "applepenapple" 可以被拆分成 "apple pen apple"。
     注意你可以重复使用字典中的单词。

                      示例 3：

                      输入: s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
输出: false

                      思路：

                      ​ 回溯，传入开始的索引，枚举选哪个，然后将索引值+1继续递归，直到递归到字符串结尾。因为有大量的重复计算，添加数组进行缓存做记忆化搜索，表示已i为开始的子字符串是否能成功分隔。

                      class Solution {
    char[] digits;
    Set<String> workDict;
    Boolean[] cache;

    public boolean wordBreak(String s, List<String> wordDict) {
        digits = s.toCharArray();
        this.workDict = new HashSet<>(wordDict);
        cache = new Boolean[s.length()];
        return dfs(0);
    }

    boolean dfs(int i) {
        if (i == digits.length) {
            return true;
        }
        if (cache[i] != null) {
            return cache[i];
        }
        StringBuilder builder = new StringBuilder();
        for (int j = i; j < digits.length; j++) {
            builder.append(digits[j]);
            if (workDict.contains(builder.toString()) && dfs(j + 1)) {
                cache[i] = true;
                return true;
            }
            cache[i] = false;
        }
        return false;
    }
}

                      思路： 翻译成递推，递归函数改为数组，递归改成循环，递归终止条件改为初始状态dp[s.length] = true

                      class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> workDict = new HashSet<>(wordDict);
        boolean[] dp = new boolean[s.length() + 1];
        dp[s.length()] = true;
        for (int i = s.length() - 1; i >= 0; i--) {
            StringBuilder builder = new StringBuilder();
            for (int j = i; j < s.length(); j++) {
                builder.append(s.charAt(j));
                if (workDict.contains(builder.toString()) && dp[j + 1]) {
                    dp[i] = true;
                    break;
                }
            }
        }
        return dp[0];
    }
}

                      # 306. 累加数open in new window

                      累加数 是一个字符串，组成它的数字可以形成累加序列。

                      一个有效的 累加序列 必须 至少 包含 3 个数。除了最开始的两个数以外，序列中的每个后续数字必须是它之前两个数字之和。

                      给你一个只包含数字 '0'-'9' 的字符串，编写一个算法来判断给定输入是否是 累加数 。如果是，返回 true ；否则，返回 false 。

                      **说明：**累加序列里的数，除数字 0 之外，不会 以 0 开头，所以不会出现 1, 2, 03 或者 1, 02, 3 的情况。

                      示例 1：

                      输入："112358"
输出：true 
解释：累加序列为: 1, 1, 2, 3, 5, 8 。1 + 1 = 2, 1 + 2 = 3, 2 + 3 = 5, 3 + 5 = 8

                      示例 2：

                      输入："199100199"
输出：true 
解释：累加序列为: 1, 99, 100, 199。1 + 99 = 100, 99 + 100 = 199

                      提示：

                      • 1 <= num.length <= 35
                      • num 仅由数字（0 - 9）组成

                      进阶：你计划如何处理由过大的整数输入导致的溢出?

                      思路：

                      ​ 选或不选

                      class Solution {
    List<Long> path = new ArrayList<>();
    String num;

    public boolean isAdditiveNumber(String num) {
        this.num = num;
        return dfs(0, 0);
    }

    boolean dfs(int i, long x) {
        if (path.size() >= 3 && path.get(path.size() - 3) + path.get(path.size() - 2) != path.getLast()) {
            return false;
        }
        if (i == num.length()) {
            if (x != 0) {
                if (path.size() < 2) return false;
                return path.get(path.size() - 2) + path.getLast() == x;
            } else {
                return path.size() >= 3;
            }
        }
        x = x * 10 + num.charAt(i) - '0';
        path.add(x);
        if (dfs(i + 1, 0)) {
            return true;
        }
        path.removeLast();
        if (x != 0) {
            return dfs(i + 1, x);
        }
        return false;
    }
}

                      # 784. 字母大小写全排列open in new window

                      给定一个字符串 s ，通过将字符串 s 中的每个字母转变大小写，我们可以获得一个新的字符串。

                      给你一个正整数 n ，请你返回 n 的 惩罚数 。

                      n 的 惩罚数 定义为所有满足以下条件 i 的数的平方和：

                      • 1 <= i <= n
                      • i * i 的十进制表示的字符串可以分割成若干连续子字符串，且这些子字符串对应的整数值之和等于 i 。

                      示例 1：

                      输入：n = 10
输出：182
解释：总共有 3 个整数 i 满足要求：
- 1 ，因为 1 * 1 = 1
- 9 ，因为 9 * 9 = 81 ，且 81 可以分割成 8 + 1 。
- 10 ，因为 10 * 10 = 100 ，且 100 可以分割成 10 + 0 。
因此，10 的惩罚数为 1 + 81 + 100 = 182

                      示例 2：

                      输入：n = 37
输出：1478
解释：总共有 4 个整数 i 满足要求：
- 1 ，因为 1 * 1 = 1
- 9 ，因为 9 * 9 = 81 ，且 81 可以分割成 8 + 1 。
- 10 ，因为 10 * 10 = 100 ，且 100 可以分割成 10 + 0 。
- 36 ，因为 36 * 36 = 1296 ，且 1296 可以分割成 1 + 29 + 6 。
因此，37 的惩罚数为 1 + 81 + 100 + 1296 = 1478

                      提示：

                      • 1 <= n <= 1000

                      思路：

                      ​ 对每个数进行判断是否满足惩罚数要求，可以使用回溯计算。

                      ​ 当前操作：选取[start, j)的子串，累加

                      ​ 子问题：对>= j的子串进行选取与累加

                      ​ 下一个子问题：对>= j + ?的子串进行选取与累加

                      class Solution {
    public int punishmentNumber(int n) {
        int res = 0;
        for (int i = 1; i <= n; i++) {
            int pow = i * i;
            if (dfs(String.valueOf(pow), i, 0, 0)) {
                res += pow;
            }
        }
        return res;
    }

    boolean dfs(String pow, int i, int start, int result) {
        if (start == pow.length()) {
            return result == i;
        }
        for (int j = start + 1; j <= pow.length(); j++) {
            int temp = Integer.parseInt(pow.substring(start, j));
            if (result + temp > i) {
                continue;
            }
            if (dfs(pow, i, j, result + temp)) {
                return true;
            }
        }
        return false;
    }
}

                      思路：

                      ​ 将删除操作转化为对每个字符的选或不选问题，注意要求删除次数最少，所以只能先选再不选，保证第一次达成答案时，选取的字符最长，即删除的字符最少。

                      ​ 当前问题：每一个位置i选或者不选

                      ​ 子问题：构造下标>= i的字符串

                      ​ 下一个子问题：构造下标>= i + 1的字符串

                      class Solution {
    Set<String> res = new HashSet<>();
    StringBuilder path = new StringBuilder();
    char[] digits;
    int max = 0;

    public List<String> removeInvalidParentheses(String s) {
        digits = s.toCharArray();
        dfs(0, 0, 0);
        return new ArrayList<>(res);
    }

    void dfs(int left, int right, int i) {
        if (i == digits.length) {
            if (left == right) {
                max = Math.max(max, path.length());
                if (path.length() == max) {
                    res.add(path.toString());
                }
            }
            return;
        }
        // 剪枝，剩余字符数不足需要的右括号个数
        if (left - right > digits.length - i) {
            return;
        }

        char c = digits[i];
        if (Character.isLowerCase(c)) {
            path.append(c);
            dfs(left, right, i + 1);
            path.deleteCharAt(path.length() - 1);
        } else if (c == '(') {
            path.append(c);
            dfs(left + 1, right, i + 1);
            path.deleteCharAt(path.length() - 1);
        } else if (left > right) {
            path.append(c);
            dfs(left, right + 1, i + 1);
            path.deleteCharAt(path.length() - 1);
        }
        dfs(left, right, i + 1);
    }
}

                      class Solution {

    String word;
    int m;

    public boolean exist(char[][] board, String word) {
        this.board = board;
        this.word = word;

    char[][] board;
    boolean[][] used;

    StringBuilder path = new StringBuilder();
    Set<String> set = new HashSet<>();
    int m;
    int n;
    int maxLength;

    public List<String> findWords(char[][] board, String[] words) {
        this.board = board;
        m = board.length;
        n = board[0].length;
        used = new boolean[m][n];
        for (String word : words) {

            maxLength = Math.max(word.length(), maxLength);

            return;

        for (int[] direct : directs) {
            int dx = x + direct[0], dy = y + direct[1];
            if (dx < 0 || dy < 0 || dx >= m || dy >= n || used[dx][dy]) {
                continue;
            }
            used[dx][dy] = true;
            path.append(board[dx][dy]);
            dfs(dx, dy);
            path.deleteCharAt(path.length() - 1);
            used[dx][dy] = false;
        }
    }
}

                      思路：

                      ​ 使用每个索引包含的字母做进一步剪枝。

                      class Solution {
    int[][] directs = new int[][]{{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
    char[][] board;
    boolean[][] used;
    List<String> res = new ArrayList<>();
    StringBuilder path = new StringBuilder();
    Set<String> set = new HashSet<>();
    Set<Character> empty = new HashSet<>();
    Map<Integer, Set<Character>> map = new HashMap<>();
    int m;
    int n;
    int maxLength;

    public List<String> findWords(char[][] board, String[] words) {
        this.board = board;
        m = board.length;
        n = board[0].length;
        used = new boolean[m][n];
        for (String word : words) {
            set.add(word);
            maxLength = Math.max(word.length(), maxLength);
            for (int i = 0; i < word.length(); i++) {
                map.computeIfAbsent(i, t -> new HashSet<>()).add(word.charAt(i));
            }
        }
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (map.get(0).contains(board[i][j])) {
                    used[i][j] = true;
                    path.append(board[i][j]);
                    dfs(i, j);
                    path.deleteCharAt(path.length() - 1);
                    used[i][j] = false;
                }
            }
        }
        return res;
    }

    void dfs(int x, int y) {
        if (path.length() > maxLength) {
            return;
        }
        if (set.contains(path.toString())) {
            res.add(path.toString());
            set.remove(path.toString());
        }
        for (int[] direct : directs) {
            int dx = x + direct[0], dy = y + direct[1];
            if (dx < 0 || dy < 0 || dx >= m || dy >= n || used[dx][dy]) {
                continue;
            }
            if (map.getOrDefault(path.length(), empty).contains(board[dx][dy])) {
                used[dx][dy] = true;
                path.append(board[dx][dy]);
                dfs(dx, dy);
                path.deleteCharAt(path.length() - 1);
                used[dx][dy] = false;
            }
        }
    }
}

                      给你一个下标从 0 开始的 m x n 二进制 矩阵 grid 。你可以从一个格子 (row, col) 移动到格子 (row + 1, col) 或者 (row, col + 1) ，前提是前往的格子值为 1 。如果从 (0, 0) 到 (m - 1, n - 1) 没有任何路径，我们称该矩阵是 不连通 的。

                      如果可以使矩阵不连通，请你返回 true ，否则返回 false 。

                      注意 ，翻转一个格子的值，可以使它的值从 0 变 1 ，或从 1 变 0 。

                      示例 1：

                      输入：grid = [[1,1,1],[1,0,0],[1,1,1]]
输出：true
解释：按照上图所示我们翻转蓝色格子里的值，翻转后从 (0, 0) 到 (2, 2) 没有路径。

                      示例 2：

                      输入：grid = [[1,1,1],[1,0,1],[1,1,1]]
输出：false
解释：无法翻转至多一个格子，使 (0, 0) 到 (2, 2) 没有路径。

                      提示：

                      • m == grid.length
                      • n == grid[i].length
                      • 1 <= m, n <= 1000
                      • 1 <= m * n <= 10^5
                      • grid[0][0] == grid[m - 1][n - 1] == 1

                      思路：

                      ​ 找到边界，上轮廓或者下轮廓，如果轮廓去除了点，别的路径走不通，则不联通。

                      class Solution {
    private int[][] grid;
    private int m, n;

    public boolean isPossibleToCutPath(int[][] grid) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        return !dfs(0, 0) || !dfs(0, 0);
    }

    boolean dfs(int x, int y) {
        if (x == m - 1 && y == n - 1) return true;
        grid[x][y] = 0;
        return x < m - 1 && grid[x + 1][y] == 1 && dfs(x + 1, y) ||
                y < n - 1 && grid[x][y + 1] == 1 && dfs(x, y + 1);
    }
}

                      # 854. 相似度为 K 的字符串open in new window

                      对于某些非负整数 k ，如果交换 s1 中两个字母的位置恰好 k 次，能够使结果字符串等于 s2 ，则认为字符串 s1 和 s2 的 相似度为 k 。

                      给你两个字母异位词 s1 和 s2 ，返回 s1 和 s2 的相似度 k 的最小值。

                      示例 1：

                      输入：s1 = "ab", s2 = "ba"
输出：1

                      示例 2：

                      输入：s1 = "abc", s2 = "bca"
输出：2

                      提示：

                      • 1 <= s1.length <= 20
                      • s2.length == s1.length
                      • s1 和 s2 只包含集合 {'a', 'b', 'c', 'd', 'e', 'f'} 中的小写字母
                      • s2 是 s1 的一个字母异位词

                      思路：

                      ​ dfs回溯比较。

                      class Solution {
    int result = Integer.MAX_VALUE;

    public int kSimilarity(String s1, String s2) {
        return dfs(s1.toCharArray(), s2.toCharArray(), 0, 0);
    }

    int dfs(char[] a, char[] b, int start, int current) {
        // 设置递归终止条件 当前的交换次数已经大于等于当前交换次数的最小值，就没必要再往下交换了，直接返回
        if (current >= result) return result;
        // 当走到了末尾，即完成了交换，取交换次数的最小值
        if (start == a.length - 1) return result = current;

        for (int i = start; i < a.length; i++) {
            if (a[i] != b[i]) {
                for (int j = i + 1; j < b.length; j++) {
                    /* 这里为什么加a[j]!=b[j]呢？
                    举个例子:
                    s1 = "aacba"
                    一种交换情况：s2 = "baaca" --> "abaca" --> "aabca" --> "aacba"
                    另一种情况：  s2 = "baaca" --> "aabca" --> "aacba"
                    为什么第二种情况优于第一种？
                    因为第二种情况，s1[1] == s2[1] == 'a'，所以保持不动，找下一个'a'值(保证a[j] != b[j]即可)进行交换，进而减少交换的次数 */
                    if (b[j] == a[i] && b[j] != a[j]) {
                        // 暂时交换
                        swap(b, i, j);
                        // 交换之后，开始对比的位置即可往下移一位(i + 1)
                        dfs(a, b, i + 1, current + 1);
                        // 上面的交换次数过大或者对换完了就回溯(回到原来的位置)
                        swap(b, i, j);
                        // 如果字符串a、b数组当前所交换的i、j值互为相等，那么交换一次即可匹配两个字符位，这是我们理想情况，
                        // 故后面的字符不要遍历下去，此时跳出当层循环
                        if (b[i] == a[j]) break;
                    }
                }
                // 这里为什么要返回res呢？
                // 这里return res表示当前修改路径的一个值，用来结束递归方法，
                // 如果这里在得到正确的顺序不结束递归的话，那么可能会导致回溯之后的字符串，又接着i自增往下一个位置递归，而上一个位置却出现同一位置字母不同的情况
                return result;
            }
        }
        // 表示后面没有可以修改的点,并且最小值一定要存到res中
        return result = current;
    }

    void swap(char[] sc, int i, int j) {
        char temp = sc[i];
        sc[i] = sc[j];
        sc[j] = temp;
    }
}

                      # 2477. 到达首都的最少油耗open in new window

                      给你一棵 n 个节点的树（一个无向、连通、无环图），每个节点表示一个城市，编号从 0 到 n - 1 ，且恰好有 n - 1 条路。0 是首都。给你一个二维整数数组 roads ，其中 roads[i] = [ai, bi] ，表示城市 ai 和 bi 之间有一条 双向路 。

                      提示：

                      • roads.length == n - 1

                      • ai != bi
                      • roads 表示一棵合法的树。
                      • 1 <= seats <= 105

                      思路：

                      ​ 统计每一个节点上所有人到上一级节点的油耗，进行累加。

                      class Solution {
    private long result;

    public long minimumFuelCost(int[][] roads, int seats) {
        HashMap<Integer, List<Integer>> map = new HashMap<>();
        for (int[] road : roads) {
            map.computeIfAbsent(road[0], t -> new ArrayList<>()).add(road[1]);
            map.computeIfAbsent(road[1], t -> new ArrayList<>()).add(road[0]);
        }
        for (int i : map.getOrDefault(0, List.of())) {
            dfs(i, 0, map, seats);
        }
        return result;
    }
    
    private int dfs(int n, int from, HashMap<Integer, List<Integer>> map, int seats) {
        int count = 1;
        for (int i : map.get(n)) {
            count += i == from ? 0 : dfs(i, n, map, seats);
        }
        result += (count + seats - 1) / seats; // 这里加的是当前节点所有人去往上一节点的油耗
        // 向上取整，原本是 (t) / seats，假如t=1, seats为5，这个结果是0。若想向上取整并不影响原来结果，最少得加上4，也就是seats-1。（加seats的话 例如：0/5，结果就变了 ）
        return count ; // 这里返回的是当前节点去往上一节点的人数
    }
}
