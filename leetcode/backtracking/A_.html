          # 题目

          # 854. 相似度为 K 的字符串open in new window

          对于某些非负整数 k ，如果交换 s1 中两个字母的位置恰好 k 次，能够使结果字符串等于 s2 ，则认为字符串 s1 和 s2 的 相似度为 k 。

          给你两个字母异位词 s1 和 s2 ，返回 s1 和 s2 的相似度 k 的最小值。

          示例 1：

          输入：s1 = "ab", s2 = "ba"
输出：1

          示例 2：

          输入：s1 = "abc", s2 = "bca"
输出：2

          提示：

          • 1 <= s1.length <= 20
          • s2.length == s1.length
          • s1 和 s2 只包含集合 {'a', 'b', 'c', 'd', 'e', 'f'} 中的小写字母
          • s2 是 s1 的一个字母异位词

          思路：

          ​ A-star启发式搜索

          class Solution {
    public int kSimilarity(String s1, String s2) {
        if (s1.equals(s2)) return 0;
        int res = 0;
        Map<String, Integer> g = new HashMap<>();
        g.put(s1, 0);
        PriorityQueue<Map.Entry<String, Integer>> queue = new PriorityQueue<>(Map.Entry.comparingByValue());
        queue.add(Map.entry(s1, g.get(s1) + f(s1, s2)));
        while (!queue.isEmpty()) {
            String key = queue.poll().getKey();
            if (key.equals(s2)) return g.get(key);
            List<String> nexts = getNext(key, s2);
            for (String next : nexts) {
                // 经过 g[key]+1 步可以走到 next 状态，如果此前走到 next 状态的步数 g[next] 比 g[key]+1 大，那么我们应该取较小的步数，也就是取 g[key]+1
                if (!g.containsKey(next) || g.get(next) > g.get(key) + 1) {
                    g.put(next, g.get(key) + 1);
                    queue.offer(Map.entry(next, g.get(next) + f(next, s2)));
                }
            }
        }
        return -1;
    }

    int f(String cur, String target) {
        int res = 0;
        for (int i = 0; i < cur.length(); i++) {
            if (cur.charAt(i) != target.charAt(i)) res++;
        }
        // 假设有 7 个不同的位置，一次交换最多可以消除 2 个不同，需要4次。8个也需要4次，但(8+1)/2仍然是4，所以是(res+1)/2
        return (res + 1) / 2;
    }

    List<String> getNext(String cur, String t) {
        List<String> nexts = new ArrayList<>();
        char[] current = cur.toCharArray(), target = t.toCharArray();
        int idx = 0, length = current.length;
        for (int i = 0; i < length; i++) {
            if (current[i] != target[i]) {
                idx = i;
                break;
            }
        }
        for (int i = idx + 1; i < length; i++) {
            if (current[i] == target[idx] && current[i] != target[i]) {
                swap(current, idx, i);
                nexts.add(String.valueOf(current));
                swap(current, idx, i);
            }
        }
        return nexts;
    }


    void swap(char[] array, int i, int j) {
        char temp = array[i];
        array[i] = array[j];
        array[j] = temp;
    }
}
